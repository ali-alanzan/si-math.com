# ⚡ دليل البدء السريع - 5 دقائق

## 🎯 الهدف
تشغيل نظام الموديلات المستقل في أقل من 5 دقائق!

---

## ✅ الخطوات

### 1️⃣ رفع الملفات (دقيقة واحدة)

```bash
/wp-content/plugins/advanced-quiz-system-v4/
├── includes/
│   ├── class-module-manager-standalone.php   ← جديد ✨
│   ├── class-adaptive-quiz.php              ← استبدل
│   ├── class-chat-system.php                ← استبدل
│   └── class-drawing-canvas.php             ← استبدل
│
└── assets/js/
    ├── module-admin.js    ← جديد ✨
    └── module-frontend.js ← جديد ✨
```

### 2️⃣ قاعدة البيانات (30 ثانية)

افتح `phpMyAdmin` ونفذ:

```sql
CREATE TABLE IF NOT EXISTS `wp_aqs_module_attempts` (
  `id` bigint(20) NOT NULL AUTO_INCREMENT,
  `user_id` bigint(20) NOT NULL,
  `module_id` bigint(20) NOT NULL,
  `course_id` bigint(20) NOT NULL,
  `score` decimal(5,2) NOT NULL,
  `total_questions` int(11) NOT NULL,
  `correct_answers` int(11) NOT NULL,
  `created_at` datetime NOT NULL,
  PRIMARY KEY (`id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

### 3️⃣ تفعيل الكلاس (30 ثانية)

في ملف البلاجن الرئيسي (مثل `advanced-quiz-system.php`)، أضف:

```php
// بعد السطر الأخير من require_once
require_once plugin_dir_path(__FILE__) . 'includes/class-module-manager-standalone.php';

// في نهاية الملف
add_action('plugins_loaded', function() {
    AQS_Module_Manager::get_instance();
});
```

### 4️⃣ Flush Permalinks (10 ثواني)

اذهب إلى: `لوحة التحكم → الإعدادات → Permalinks`  
اضغط **"حفظ التغييرات"**

### 5️⃣ اختبر! (دقيقتين)

1. **في Dashboard:**
   - شوف القائمة الجديدة: `📚 الموديلات التكيفية`
   - اضغط "إضافة جديد"

2. **أنشئ موديل تجريبي:**
   - العنوان: `Test Module`
   - الكورس: اختر أي كورس
   - النوع: Module 1
   - أضف سؤالين سريعين
   - احفظ ✅

3. **اختبر في Frontend:**
   - افتح الكورس كطالب
   - شوف الموديل ظاهر في الأسفل
   - اضغط "ابدأ الامتحان"
   - حل الأسئلة
   - شوف النتيجة! 🎉

## 🚨 مشاكل شائعة

| المشكلة | الحل الفوري |
|---------|-------------|
| ما أشوف القائمة | تأكد من تفعيل الكلاس في الملف الرئيسي |
| الموديل ما يظهر للطلاب | تأكد الطالب مسجل في الكورس |
| 404 Error | روح `الإعدادات → Permalinks` واضغط حفظ |
| JavaScript لا يعمل | افتح F12 وشوف الأخطاء |

## 📋 الاستخدام اليومي

### إنشاء موديل جديد:
1. اذهب إلى `الموديلات التكيفية → إضافة جديد`
2. املأ البيانات الأساسية
3. أضف الأسئلة
4. احفظ → **تلقائياً يظهر للطلاب!** ✨

### عرض جميع الموديلات:
`لوحة التحكم → الموديلات التكيفية → جميع الموديلات`

### تعديل موديل:
اضغط "✏️ تعديل" بجانب أي موديل

## 🎓 مثال سريع

**السيناريو:** كورس رياضيات

**Module 1 - تقييم المستوى:**
```
العنوان: اختبار تحديد المستوى
الكورس: رياضيات للمبتدئين
النوع: Module 1
الأسئلة:
  1. كم 5 + 3؟ (سهل)
  2. كم 12 × 7؟ (صعب)
  3. ما ناتج 15 - 8؟ (سهل)
```

**Module 2 - امتحان تكيفي:**
```
العنوان: الامتحان النهائي
الكورس: رياضيات للمبتدئين
النوع: Module 2
الأسئلة:
  - للضعفاء: 10 أسئلة سهلة (جمع/طرح)
  - للأقوياء: 10 أسئلة صعبة (ضرب/قسمة)
```

**النتيجة:**
- الطالب يحل Module 1
- إذا جاب أقل من 50% → Module 2 يعرض الأسئلة السهلة
- إذا جاب أكثر من 50% → Module 2 يعرض الأسئلة الصعبة

## 🎨 التخصيص

### تغيير نسبة التكيف:
في `wp-config.php` أو Admin Settings:
```php
update_option('aqs_easy_threshold', 60); // الافتراضي: 50
```

### تغيير الألوان:
عدّل في `module-frontend.js` السطر 200+

## 🔥 نصيحة ذهبية

**لأفضل تجربة:**
1. اجعل Module 1 متوسط الصعوبة (خليط)

3. حدد وقت مناسب (1.5 دقيقة/سؤال)
4. اختبر بنفسك قبل ما تنشر!
