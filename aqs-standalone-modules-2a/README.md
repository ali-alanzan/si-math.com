# 🚀 نظام الموديلات المستقل - دليل التثبيت الكامل

## 🎯 نظرة عامة

هذا النظام يتيح لك إنشاء وإدارة الموديلات التكيفية من **مكان واحد مستقل تماماً**!

### ✨ الميزات الرئيسية:

1. **صفحة Admin مستقلة تماماً** - قائمة خاصة في Dashboard
2. **إنشاء موديلات بدون الحاجة للكويزات** - كل شيء من مكان واحد
3. **يظهر تلقائياً للطلاب** - ربط تلقائي بالكورسات
4. **نظام أسئلة متطور** - أسئلة متعددة الاختيارات، صح/خطأ، مستويات صعوبة
5. **تكيف ذكي** - Module 2 يتكيف مع أداء الطالب في Module 1
6. **تايمر وتتبع** - حساب الوقت والدرجات تلقائياً

## 🔧 خطوات التثبيت

### الخطوة 1: رفع الملفات

**الطريقة الأولى: استبدال الملفات القديمة**
```bash
/wp-content/plugins/advanced-quiz-system-v4/includes/
├── class-module-manager-standalone.php   ← أضف جديد
├── class-adaptive-quiz.php              ← استبدل
├── class-chat-system.php                ← استبدل
└── class-drawing-canvas.php             ← استبدل

/wp-content/plugins/advanced-quiz-system-v4/assets/js/
├── module-admin.js      ← أضف جديد
└── module-frontend.js   ← أضف جديد
```

**الطريقة الثانية: إنشاء ملحق منفصل**
1. أنشئ مجلد جديد: `/wp-content/plugins/aqs-standalone-modules/`
2. انسخ كل الملفات من هذا الباكدج
3. أنشئ ملف `aqs-standalone-modules.php` (الكود في الأسفل)

### الخطوة 2: تشغيل قاعدة البيانات

افتح `phpMyAdmin` ونفذ الكود الموجود في `DATABASE_TABLES.sql`:

```sql
CREATE TABLE IF NOT EXISTS `wp_aqs_module_attempts` (
  -- ... انسخ الكود من DATABASE_TABLES.sql
);
```

**أو** استخدم WP-CLI:
```bash
wp db query < DATABASE_TABLES.sql
```

### الخطوة 3: تفعيل الكلاس الجديد

في ملف البلاجن الرئيسي (`advanced-quiz-system.php` أو مشابه)، أضف:

```php
// بعد تحميل الكلاسات الأخرى:
require_once plugin_dir_path(__FILE__) . 'includes/class-module-manager-standalone.php';

// Initialize
add_action('plugins_loaded', function() {
    AQS_Module_Manager::get_instance();
});
```

### الخطوة 4: Flush Permalinks

1. اذهب إلى `لوحة التحكم → الإعدادات → Permalinks`
2. اضغط **"حفظ التغييرات"**

## 🎨 كيفية الاستخدام

### 1. الوصول للنظام

بعد التثبيت، ستجد قائمة جديدة في Dashboard:

```
📚 الموديلات التكيفية
  ├── جميع الموديلات
  └── إضافة جديد
```

### 2. إنشاء موديل جديد

**أ. اذهب إلى "إضافة جديد"**

**ب. املأ المعلومات الأساسية:**
- عنوان الموديل: `Module 1 - امتحان تحديد المستوى`
- الكورس المرتبط: اختر الكورس
- نوع الموديل:
  - **Module 1** (الأساسي) - لتحديد مستوى الطالب
  - **Module 2** (التكيفي) - يتكيف مع الأداء
- نسبة النجاح: `50%`
- الوقت المحدد: `30` دقيقة (أو `0` لبدون حد)

**ج. إضافة الأسئلة:**

1. اضغط **"➕ إضافة سؤال جديد"**
2. اكتب نص السؤال
3. اختر نوع السؤال (متعدد الاختيارات / صح وخطأ)
4. حدد مستوى الصعوبة (سهل / صعب)
5. أضف الإجابات
6. اختر الإجابة الصحيحة (بالراديو بوتن)

**د. احفظ الموديل**

### 3. عرض الموديلات للطلاب

الموديلات **تظهر تلقائياً** في صفحة الكورس للطلاب المسجلين!

```
┌─────────────────────────────────┐
│   📝 الامتحانات التكيفية      │
├─────────────────────────────────┤
│  🎯 Module 1                    │
│  امتحان تحديد المستوى          │
│  📊 10 أسئلة | ⏱️ 30 دقيقة    │
│  [ابدأ الامتحان →]             │
└─────────────────────────────────┘
```

## 🧠 كيف يعمل النظام التكيفي؟

### Module 1 (الأساسي):
1. الطالب يحل Module 1
2. النظام يسجل الدرجة
3. إذا الدرجة < 50% → الطالب "ضعيف"
4. إذا الدرجة ≥ 50% → الطالب "جيد"

### Module 2 (التكيفي):
1. الطالب يبدأ Module 2
2. النظام يتحقق من درجة Module 1
3. **إذا كان ضعيفاً:** يعرض الأسئلة السهلة فقط
4. **إذا كان جيداً:** يعرض الأسئلة الصعبة فقط

## 🐛 استكشاف الأخطاء

### المشكلة: لا أرى القائمة الجديدة في Dashboard

**الحل:**
1. تأكد من تفعيل الكلاس في الملف الرئيسي
2. تأكد من صلاحيات المستخدم (يجب أن يكون Admin)
3. حاول تعطيل وتفعيل البلاجن

### المشكلة: الموديلات لا تظهر للطلاب

**الحل:**
1. تأكد من أن الطالب **مسجل في الكورس**
2. تأكد من نشر الموديل (Status = منشور)
3. تأكد من ربط الموديل بالكورس الصحيح

### المشكلة: الموديلات تعطي 404

**الحل:**
1. اذهب إلى صفحة "الموديلات التكيفية"
2. اضغط **"🔄 إعادة تحميل الروابط"**
3. أو: `لوحة التحكم → الإعدادات → Permalinks` واضغط حفظ

### المشكلة: Module 2 يقول "يجب إكمال Module 1"

**الحل:** هذا طبيعي! يجب على الطالب حل Module 1 أولاً.

### المشكلة: JavaScript لا يعمل

**الحل:**
1. تأكد من نسخ ملفات `assets/js/`
2. افتح Console (F12) وشوف الأخطاء
3. تأكد من تحميل jQuery

## 📈 قاعدة البيانات

### جدول wp_aqs_module_attempts

يحفظ جميع محاولات الطلاب:

```
id | user_id | module_id | course_id | score | correct | total | created_at
---|---------|-----------|-----------|-------|---------|-------|------------
1  | 5       | 12        | 8         | 75.00 | 15      | 20    | 2025-01-31
2  | 5       | 13        | 8         | 82.00 | 12      | 15    | 2025-01-31
```

### الاستعلامات المفيدة:

```sql
-- جميع محاولات طالب معين
SELECT * FROM wp_aqs_module_attempts WHERE user_id = 5;

-- متوسط الدرجات لموديل معين
SELECT AVG(score) FROM wp_aqs_module_attempts WHERE module_id = 12;

-- أفضل 10 طلاب
SELECT user_id, AVG(score) as avg_score 
FROM wp_aqs_module_attempts 
GROUP BY user_id 
ORDER BY avg_score DESC 
LIMIT 10;
```

## 🎓 نصائح للاستخدام الأمثل

- **Module 1:** أسئلة متنوعة (سهلة + متوسطة)

### 3. نسبة النجاح:
- **Module 1:** 40-50% (لتصنيف دقيق)
