## 🔧 التثبيت

### المتطلبات:
- WordPress 5.5 أو أحدث
- PHP 7.4 أو أحدث
- Tutor LMS Plugin
- MySQL 5.7 أو أحدث

### خطوات التثبيت:

1. **رفع البلاجن:**
   - اذهب إلى: لوحة التحكم > إضافات > إضافة جديد > رفع إضافة
   - اختر ملف الـ ZIP
   - اضغط "التثبيت الآن"

2. **تفعيل البلاجن:**
   - بعد التثبيت، اضغط "تفعيل"

3. **إعداد البلاجن:**
   - اذهب إلى: لوحة التحكم > Advanced Quiz System
   - راجع الإعدادات وفعّل المميزات المطلوبة

## 📖 دليل الاستخدام السريع

### للطلاب:

#### استخدام الدردشة العامة:
1. انقر على أيقونة الدردشة 💬 في أسفل الصفحة
2. اكتب رسالتك
3. اضغط "إرسال"

#### استخدام الرسائل الخاصة (جديد!):
1. افتح نافذة الدردشة
2. اضغط على تبويب "الرسائل الخاصة" 🔒
3. اختر الطالب من القائمة
4. ابدأ المحادثة الخاصة

#### استخدام آلة Desmos (جديد!):
1. انقر على زر "الآلة الحاسبة" 📊 في أسفل اليسار
2. أدخل المعادلات
3. شاهد الرسم البياني
4. استخدم الماوس للتكبير/التصغير
