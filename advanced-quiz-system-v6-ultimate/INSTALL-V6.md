# دليل التثبيت السريع - V6 Ultimate 🚀

## خطوات التثبيت (5 دقائق)

### 1. رفع البلاجن
- لوحة التحكم > إضافات > رفع إضافة
- اختر الملف واضغط تثبيت

### 2. تفعيل
- اضغط "تفعيل الإضافة"

### 3. الإعدادات
```
Advanced Quiz System > Settings
✅ فعّل الدردشة
✅ فعّل الرسائل الخاصة (NEW!)
✅ فعّل Desmos (NEW!)
```

## المميزات الجديدة

### 💬 رسائل خاصة
افتح الدردشة > تبويب "رسائل خاصة" > اختر طالب

### 📊 Desmos
اضغط زر الحاسبة > اكتب المعادلة > شاهد الرسم!
